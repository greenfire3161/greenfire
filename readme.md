# 🌿 Green Fire SRL - Sistema de Control de Gestión (SCG)

Este repositorio contiene el "cerebro" operativo de Green Fire. 
El objetivo es centralizar la producción de **Colonia San José** con la 
administración y ventas del **Local Santa Fe Centro**.

## 🚀 Módulos del Sistema

1. **Fábrica (Tobías):** Control de preparación de pedidos y stock real.
2. **Administración (Nicolás):** Carga de costos vivos, insumos y gastos fijos.
3. **Comercial (Rocío):** Gestión de ventas minoristas y logística de Mercado Libre.
4. **Tablero de Control (Nahuel):** Visualización de rentabilidad neta en tiempo real.

## 📈 Parámetros Críticos de Negocio
* **Costo de Estructura Mensual:** $19.855.000
* **Capacidad de Producción Objetivo:** 150.000 Litros/mes.
* **Producto Estrella:** Champicompost 25L.
